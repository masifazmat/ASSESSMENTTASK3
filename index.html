Task 3

Name Web Attacks
SQL Injection

Source: SQL Injection

 An attack technique where malicious SQL code is inserted into input fields to manipulate or query the database directly. This can lead to unauthorized data access
Source SQL_Injection Details
Cross-Site Scripting

Source: Cross Site Scripting

 Attackers inject malicious scripts into web pages viewed by users. The script can then be used to steal information or perform actions on behalf of the user.
Source: Cross Site Scripting Details
Distributed Denial of Service (DDoS)

Source: DDoS

 Overwhelming a system with a flood of traffic, rendering it unavailable to legitimate users.
Source: DDoS Details
Man-in-the-Middle (MitM) Attack

Source: MitM Attack

 An attacker secretly intercepts and potentially alters the communication between two parties to steal information or inject malicious content.
Source: MitM Attack Details
Session Hijacking

Source: Session Hijacking

 An attacker exploits a valid session ID to gain unauthorized access to a web application, allowing them to impersonate the victimized user.
Source: Session Hijacking Details
XML External Entity (XXE)

Source: XXE

 An attack where an attacker can exploit vulnerable XML processors to execute arbitrary code, access local files, or perform other malicious activities.
Source: XXE Details
Remote File Inclusion (RFI)

Source: RFI

 An attacker uses this technique to include remote files, typically through manipulating input data. This can allow for code execution on the server.
Source: RFI Details
Clickjacking

Source: Clickjacking

 A malicious technique of tricking a user into clicking on something different from what the user perceives. This can be used to steal information or take control of actions.
Source: Clickjacking Details
